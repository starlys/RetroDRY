RetroDRY skeletal demo and integration test runner

This is written in plain javascript with no bundling/transpiling/etc, so you can see everything without layers of complexity.

Run Samples

When you click the button, nothing will appear to happen. View the browser console for output, and also view main.js for the functions that get run here.

Integration tests

Open browser console to see integration test output.

Quick test suite

The quick test suite runs tests sequentially with a single client and doesn't look at any timing or multi-user effects. It clears the database to start and leaves records in there after it is done.

Slow test suite

The slow test suite connects multiple clients and tests subscriptions, timing, abandoned sessions, and other more complex cases. Watch the console to view progress. It clears the database to start and leaves records in there after it is done.
